# 技能模板

这是一个用于创建新技能的模板文件。请根据您的具体需求修改此文件的内容。

## 使用方法

1. 复制此模板文件
2. 重命名为您的技能名称
3. 更新YAML元数据部分
4. 编写技能的具体描述和功能说明

## 技能结构

一个完整的技能文件应包含以下部分：

- **YAML元数据**: 技能的基本信息
- **技能描述**: 详细的功能说明和使用方法
- **功能列表**: 技能提供的具体功能
- **使用示例**: 如何调用和使用技能
- **注意事项**: 使用技能时的注意事项

请根据您的实际需求填充以上内容。
